# Bali Rent WordPress Theme

Custom WordPress theme for vehicle rental business (motorbike & car rental in Bali).

## Features
- Custom Post Type (Vehicles, Testimonials, FAQ)
- ACF Integration (Hero, Pricing, Specs)
- WhatsApp Booking Integration
- Responsive Design
- Dynamic Gallery (JS)

## Tech Stack
- WordPress (Custom Theme)
- PHP
- ACF (Free)
- Vanilla JS

## Installation
- Download the ZIP file.
- Upload it to WordPress (Appearance → Themes → Add New → Upload Theme).
- Activate the theme.
